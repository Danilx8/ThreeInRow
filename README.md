# Три в ряд

Платформа - .NET 8.0
dotnet build -c Release
dotnet bin/Release/net8.0/ThreeInRow.dll

Ввод хода: -p 1:1,1:2
Ввод бонуса удаления по типу: -t 1:1
Ввод бонуса удаления по ряду: -l 1:1,1:2
Просмотр статистики: -s
Выход: -e
